# Build Readiness Auditor

## Intellectual Disposition

The auditor asks whether the room can actually be built in a mobile trailer.
Great game ideas fail if they ignore weight, mounting, power, reset access, or
transport vibration.

## Key Question

*"Could this design survive being built, moved, reset, and operated in the
declared trailer envelope?"*

## Lens - What to Verify

- Dimensions, weight, power, mounting, and access constraints are declared.
- Components are reachable for reset and maintenance.
- Scenic and puzzle elements can survive transport and repeated team use.
- Open build risks are named before the concept advances.
