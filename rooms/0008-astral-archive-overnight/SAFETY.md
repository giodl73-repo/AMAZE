# Astral Archive Overnight Safety

## Egress

| Item | Requirement | Current design | Open action |
|---|---|---|---|
| Visible exit | players can identify exit at all times | exit and comfort route named in briefing | verify actual trailer layout |
| Clear egress lane | no act bins/stools in path | Z6 clear lane and reset-cart outside rule | tape-test |
| Emergency stop | operator can halt and exit | stop script, lights up, state preserved if possible | write staff card |
| Comfort interruption | 90-minute session may need restroom/medical pause | staff escort and state-card preservation | venue policy |
| Final state | no real lock-in | vault is fictional; door never locked | install review |

## Hazards

| Hazard | Source | Mitigation | Owner |
|---|---|---|---|
| Fatigue/heat/cold | 90-minute trailer duration | comfort-pause policy, temp checks, seating/lean points | ops |
| Trip | tiles/relics/stools in aisle | trays, floor marks, reset cart outside | ops |
| Pinch | astrolabe/orrery wheels | low force, finger gaps, guard ring | build |
| Electrical | optional low-voltage lights | fused supply, strain relief, manual fallback | build |
| Glare/color | spectrum filters/star field | work light, printed high-contrast marks | design/build |
| Crowding | final vault | floor marks, caller/checker/handler | ops |

## Accessibility

| Need | Design response | Test |
|---|---|---|
| Seated participation | seated roles in each act and final caller role | seated full-run simulation |
| Fatigue support | act breaks, role rotation, lean/rest surfaces | 90-minute tabletop run |
| Low vision/glare | work light and high-contrast backups | glare/color test |
| No audio-only clue | chimes/ticks are flavor only | mute run |
| Memory support | act board and physical state cards | confused-team run |
| Emergency interruption | state preserved and staff escort script | pause drill |

## Operator controls

| Control | Location | Use | Failure response |
|---|---|---|---|
| Stop script | operator clipboard | halt and exit | lights up, egress clear |
| Comfort-pause script | operator clipboard | preserve state and escort if needed | stop/hold clock per policy |
| Act acceleration cards | operator binder | recover long-session drift | grant first coordinate/date/token |
| Staff proof relics | act bins | recover earned act proof | log use |
| Manual vault card | operator clipboard | C5 final recovery | story-preserving final reveal |
| Work light | staff side | glare/low-vision support | turn on without changing answer |
