# Stormglass Station Safety

## Egress

| Item | Requirement | Current design | Open action |
|---|---|---|---|
| Visible exit | players can always identify exit | exit named in briefing and floorplan | verify with actual trailer |
| Clear aisle | no props in emergency path | 3 ft planning egress lane | tape layout test |
| Emergency stop | operator can halt run | stop script, lights up, manual door path | write staff card |
| Final state | no real lock-in | beacon is fictional; door never locked | confirm install |

## Hazards

| Hazard | Source | Mitigation | Owner |
|---|---|---|---|
| Trip | cartridges/cards dropped in aisle | trays and pouch count | ops |
| Pinch | rotating dial, slider, shutter | rounded edges, low force, finger gaps | build |
| Electrical | low-voltage LEDs/button panel | fused/approved supply, strain relief | build |
| Acrylic edge | stormglass face/overlays | rounded/sanded edges | build |
| Crowding | final console | role ritual and three-sided access | design |
| Weather theming | storm sound/light | no strobe, volume cap, work-light mode | ops |

## Accessibility

| Need | Design response | Test |
|---|---|---|
| Seated participation | caller/checker role can solve from aisle; controls in reach band | seated mock run |
| Low vision/glare | high-contrast marks, work-light mode, matte overlays | glare test |
| No audio-only clue | all storm/audio effects are flavor | mute run |
| No crawling/climbing | all mechanisms on wall/counter | floorplan review |
| Fine-motor limits | cartridges are large, keyed, low-force | dexterity test |

## Operator controls

| Control | Location | Use | Failure response |
|---|---|---|---|
| Stop script | operator clipboard | halt room and exit | lights up, door path clear |
| Manual route card | operator clipboard | recover console jam | tell fiction-preserving station override |
| Staff proof cartridges | staff pouch B/C | recover earned instrument proof | log use in PLAYTEST |
| Work light | operator side | accessibility/glare support | turn on without changing answer |
| Reset photo | clipboard | verify ready state | do not start if mismatch |

## Safety gate

| Gate | Pass? | Evidence | Required revision |
|---|---|---|---|
| Egress never blocked | Draft | floorplan egress lane | tape-test real trailer |
| Staff override exists | Draft | manual route card | timed drill |
| Low-voltage plan reviewed | Draft | build plan | electrical review |
| Accessibility alternatives declared | Draft | this file and floorplan | seated/glare test |
| No real confinement | Pass | door never locked | operator script |
