# Clockwork Greenhouse Operations

## Staffing

| Role | Responsibility | Notes |
|---|---|---|
| Primary operator | briefing, hints, safety, final release | one staff member can run seed prototype |
| Reset operator | tray/token reset during busy events | optional but recommended for first tests |
| Builder on call | valve, prism, wheel repairs | required until prototypes stabilize |

## Run clock

| Phase | Target min | Operator focus | Acceleration |
|---|---:|---|---|
| Brief/safety | 3 | exit, no-force, roles | shorten story, keep safety |
| P1 system frame | 4 | confirm four systems | name root/water/light/bloom |
| P2-P4 systems | 19 | watch parallel progress | first seed/valve/light hint |
| P5 bloom | 7 | role ritual and wheel safety | staff proof/manual bloom |
| Payoff/exit | 3 | cozy wrap and exit | immediate manual bloom |
| Buffer | 9 | variance/reset prep | hard exit script |

## Hint protocol

| Level | Use when | Style | Example |
|---|---|---|---|
| Nudge | team has right station but stalls | in-fiction redirect | "The orchid care card names the system." |
| Point | wrong subsystem for 2+ minutes | physical pointer | point to proof rail silhouette |
| Demonstrate | slow max approaching | show one false case | "This soil texture rejects that seed." |
| Accelerate | clock/safety pressure | partial answer or staff token | staff proof token after explanation |
| Bypass | component failure | manual bloom/reveal | manual bloom card and exit token |

## Failure modes

| Failure | Detection | Player-facing script | Staff action | Post-run fix |
|---|---|---|---|---|
| Missing seed token | count mismatch | "The nursery has a duplicate seed tray." | swap duplicate token | enlarge/tether |
| Valve pointer drift | home mark mismatch | "The irrigation wheel needs calibration." | give water token after explanation | tighten pointer |
| Prism glare | players avoid panel | "Use the greenhouse inspection light." | work light or printed card | matte overlay |
| Bloom wheel jam | wheel/token will not move | "The conservatory accepts the verified bloom record." | manual bloom card | bevel/guard |
| Role tag lost | reset count short | "The head gardener has a spare duty tag." | use duplicate tag | print extra |
