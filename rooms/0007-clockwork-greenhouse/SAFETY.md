# Clockwork Greenhouse Safety

## Egress

| Item | Requirement | Current design | Open action |
|---|---|---|---|
| Visible exit | players always know exit | briefing and floorplan keep exit visible | verify actual trailer |
| Clear aisle | no props in emergency path | 3 ft planning lane | tape layout |
| Emergency stop | operator can halt room | stop script, lights up, door path clear | write staff card |
| Final state | no real lock-in | flower reveal is fictional; door never locked | confirm install |

## Hazards

| Hazard | Source | Mitigation | Owner |
|---|---|---|---|
| Trip | loose seed tokens | lidded trays and count cards | ops |
| Pinch | bloom wheel/petals | finger gaps, guard ring, low force | build |
| Glare | prism/acrylic panel | matte finish, work light | build |
| Electrical | optional LED flower glow | low voltage, strain relief, fused supply | build |
| Allergens/scent | greenhouse fiction | no real soil, pollen, or scent clue | design |
| Crowding | final wheel | role ritual and clear aisle | ops |

## Accessibility

| Need | Design response | Test |
|---|---|---|
| Seated participation | caller/checker role, reachable bench/wheel controls | seated mock run |
| Low vision/glare | high-contrast labels, matte prism, work-light path | glare test |
| No audio-only clue | sound/ticking is flavor only | mute run |
| No scent-only clue | no real botanical scent clue | no-scent review |
| Fine-motor limits | oversized tokens and low-force valves | dexterity test |

## Operator controls

| Control | Location | Use | Failure response |
|---|---|---|---|
| Stop script | operator clipboard | halt and exit | lights up, door clear |
| Manual bloom card | operator clipboard | recover bloom wheel jam | deliver fiction-preserving reveal |
| Staff proof tokens | staff pouches | recover earned P2-P4 proof | log use |
| Work light | operator side | prism/glare support | turn on without changing answer |
| Reset photo | clipboard | verify ready state | do not start if mismatch |

## Safety gate

| Gate | Pass? | Evidence | Required revision |
|---|---|---|---|
| Egress never blocked | Draft | floorplan lane | tape-test real trailer |
| Staff override exists | Draft | manual bloom card | timed drill |
| Low-voltage plan reviewed | Draft | build plan | electrical review |
| Accessibility alternatives declared | Draft | this file and floorplan | seated/glare test |
| No real allergens/scent dependency | Pass | no real soil/pollen | material review |
