# The Black Box at Mile Zero - Operations

## Staffing

- Operator count: one operator for simulation and prototype.
- Required training: intro/safety script, box-state tracking, no-force intervention, manual bypass, reset checklist.
- Monitoring position: outside trailer or protected operator nook with clear view of door, central object, and utility cabinet.

## Run clock

- Intro: 3-5 minutes.
- Soft-warning: 35 minutes.
- Hard-exit: 45 minutes.
- Reset: 10 minutes target.

## Hint protocol

| Beat/stuck state | Observable signal | Hint 1: point attention | Hint 2: clarify relationship | Acceleration |
|---|---|---|---|---|
| P1 evidence search | random drawer/search behavior | "The logged evidence is all on the rail." | "Match the marks, not the object names." | name first matching mark |
| P2 windows ignored | no one changes or discusses box windows | "The box changes what it will testify to." | "Move a window, then check the rail again." | set first window |
| P3 audio stuck | replaying audio repeatedly | "The transcript is admissible evidence." | "The skip tells you which words to trust." | reveal one indexed word |
| P4 cabinet toggling | repeated switch guessing | "Evidence control hates random toggles." | "The route phrase is an order, not a label." | give first switch group |
| P5 latch/force risk | pulling, twisting, or frustration | "If it resists, it is not ready to open." | "Use caller, checker, handler." | manual safe release |

## Reset checklist

| Step | Action | Verify |
|---|---|---|
| 1 | Return evidence tags to rail in seed order. | backs/marks match reset map |
| 2 | Reset black box windows and latches. | state indicator shows seed state |
| 3 | Replace transcript cards and internal reveal artifact. | full card/artifact count |
| 4 | Reset audio player. | first playback starts correctly |
| 5 | Reset utility cabinet switches/lights. | pre-run light test passes |
| 6 | Clear floor/egress and inspect central object. | door path clear and no latch binding |

## Failure modes

| Failure | Player-facing recovery | Staff action | Post-session fix |
|---|---|---|---|
| evidence tag missing | "Dispatch recovered a duplicate." | hand in duplicate tag | replace/retether tag |
| audio recorder fails | "The court transcript is the official record." | direct to transcript-only path | charge/replace device |
| box slider jams | "The evidence unit is sticking; dispatch can read the state." | set or announce state | inspect track |
| cabinet light fails | "Indicator bulb is out; the switch state still counts." | use manual indicator card | replace LED/wire |
| final latch resists | "Do not force it; evidence control is overriding." | manual release or bypass | inspect latch before next run |

## Transport and maintenance

| Item | Before transport | After transport | Routine maintenance |
|---|---|---|---|
| black box | engage transport lock, pad moving pieces | inspect latch/window alignment | daily latch and hinge check |
| evidence rail | remove or secure tags | count and rehang tags | replace worn cards |
| audio player | power off and secure | charge/test playback | battery rotation |
| cabinet | secure door and wiring | light/switch test | inspect strain relief |

## Operator stress tests

| Scenario | Expected staff action | Hard limit | Revision if failed |
|---|---|---|---|
| fighting team monopolizes box | assign caller/checker/handler roles | no player exclusion longer than one scene | split clues away from box |
| chaotic team pulls latches | stop action and no-force reset | no repeated force after warning | improve physical stops |
| confused team searches everywhere | redirect to logged evidence | hint by P1 minute 4 | reduce scenic red herrings |
| accessibility-varied team cannot use audio | transcript-only path works | no audio-only required clue | rewrite P3 |
| late/tired group reaches hard-exit | accelerate P4/P5 safely | no rushed unsafe opening | simplify final sequence |
